Rules for using the movie SQL agent

The Movie SQL Agent lets you ask questions about movies in plain English. It converts your question into a valid SQL query and shows the results in a table. To make sure it works correctly, read the rules below before using it.

Allowed Queries

  • Ask about movies, genres, studios, release years, or ratings.
  • Write questions in normal English, for example: "Show all comedy movies."
  • You can use filters such as "after 2010" or "with audience score above 80".
  • You can sort results, for example: "Show movies sorted by profitability".

Restricted Actions

  • You cannot add, update, or delete anything in the database.
  • You cannot change the structure of the database (no create, drop, or alter commands).
  • You can only search inside the Movies table.
  • Only these columns are available: Film, Genre, Lead_Studio, Audience_Score_pc, Profitability, Rotten_Tomatoes_pc, Worldwide_Gross, and Year.

Example Questions

  • "Show all movies released in 2001."
  • "List all comedy films."
  • "Which movies have an audience score above 80?"
  • "Show movies where profitability is greater than 2.0."
  • "Display all films made by Universal Pictures."
  • "Find movies released after 2015."
  • "List movies with a Rotten Tomatoes score over 90."

Invalid Questions

  • "Delete all movies from 2010." — Not allowed, this would change the database.
  • "Add a new movie." — Not supported, the agent can only read data.
  • "Drop the Movies table." — Forbidden, that would break the database.
  • "Show all users." — There is no Users table in this database.

How the Agent Works

Every time you type a question, the agent turns it into a safe SQL query that always starts with: 

SELECT * FROM Movies ...

It only runs read-only queries, which means it never changes or deletes data. All results come directly from the Movies table in the database.

Tips for Better Results

  • Be specific when you ask a question.
  • Use clear words like "show", "list", or "find".
  • Combine filters for more precise searches, for example: "Show comedy movies from 2010".
  • If no results appear, try rephrasing your question or removing filters.
